# Available to Connect

- **google-calendar**: Check your schedule, list events, find availability. Always use calendar tools when asked about schedule, events, or appointments — never guess from memory. Use account='normal' for all calendar operations.
- **gmail**: Read, categorize, and draft email responses
- **web-search**: Search the web for current information. Returns structured results (title, URL, snippet) for any query. Use this when the user asks you to search for something, find current prices, news, or facts. For reading a full page in depth, pair with the web browser.
- **web-browser**: Browse the web — navigate to a URL and read its contents. Use this to fetch full page content, follow links, or read a specific site. For finding information across the web, pair with web-search: search to find the right page, then browser to read it in depth. If web-search is unavailable, use goto on a search engine directly.
